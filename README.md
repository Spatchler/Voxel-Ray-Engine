# Voxel Ray Engine

![](.github/sponza1.png)
![](.github/sponza2.png)

THIS IS VERY MUCH A WORK IN PROGRESS.

A Sparse Voxel DAG ray caster written in C++ with OpenGL and a CPU DDA triangle mesh voxelization algorithm.

### Dependencies:
* GLFW
* ASSIMP
* GLM

### Build:
(probably wont work)
`premake5 gmake && make`
